# 📇 Agenda de Clientes

Este é um projeto desenvolvido para fins de estudo, focado no aprendizado de **JavaScript** puro e na integração com **APIs**. A aplicação permite gerenciar uma lista de clientes, colocando em prática conceitos fundamentais de desenvolvimento web.

---

## 🚀 Funcionalidades

* **Cadastro de Clientes:** Adição de novos contatos à base de dados.
* **Listagem Dinâmica:** Visualização de todos os clientes cadastrados consumindo dados via API.
* **Gerenciamento de Dados:** Funções de visualização, edição ou exclusão (CRUD).
* **Integração com API:** Comunicação assíncrona para persistência ou busca de informações.

---

## 🛠️ Tecnologias Utilizadas

* [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript)
* [HTML5](https://developer.mozilla.org/pt-BR/docs/Web/HTML)
* [CSS3](https://developer.mozilla.org/pt-BR/docs/Web/CSS)

## 📥 Instalação e Execução

Para rodar este projeto localmente, siga os passos abaixo:

1. **Clone o repositório:**
```bash
git clone (https://github.com/Iamilyfelix/agenda-clientes-js.git)
```
2. **Acesse a pasta do projeto:**
```bash
cd agenda-clientes-js
```
3. **Execução:**
Como o projeto utiliza tecnologias fundamentais (HTML/JS), você pode simplesmente abrir o arquivo index.html diretamente no seu navegador ou utilizar a extensão Live Server do VS Code para uma melhor experiência.
